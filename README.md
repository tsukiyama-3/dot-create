# ドット絵作るアプリ

## URL
https://dot-creator.web.app/

## build command
`npm run build`

## deploy command
`firebase deploy`

## router install
`npm i vue-router@next`

## dom-to-image install
`npm i dom-to-image`

## firebase hosting
`npm install -g firebase-tools`

`firebase login`

`firebase init`
